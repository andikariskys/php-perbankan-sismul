# Informasi Tambahan Untuk Pengembangan

## Data Session 
Daftar session yang dapat digunakan dalam project ini
```php
$_SESSION['user_id'] // mengambil id user
$_SESSION['role_id'] // mengambil role id
$_SESSION['nama_role'] // mengambil nama role
$_SESSION['nama_lengkap'] // mengambil nama lengkap
$_SESSION['email'] // mengambil email
$_SESSION['foto_profil'] // mengambil foto profil
```
